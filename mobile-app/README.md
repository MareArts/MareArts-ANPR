## Key Features

- **On-device AI** - All processing happens locally (~100-160ms per scan)
- **Team Work** - Leader/member collaboration with shared data
- **Webhooks** - Send detections to Discord, Slack, or custom endpoints
- **Cloud Sync** - Sync data across devices via marearts.com
- **Whitelist/Blacklist** - Rules with sound and vibration alerts
- **CSV Export** - Export all detections and rules
- **Map View** - See all detections on a map with GPS
- **Web Rules** - Upload rules on marearts.com, download on phone
- **Vehicle Info (Beta)** - Make, model, color, type from cloud AI
- **Landscape Camera** - Full support for landscape scanning
- **100% Offline** - Works without internet (cloud features optional)

## App Tabs

| Tab | Purpose |
|-----|---------|
| **Scan** | Fast on-device plate scanning (single, continuous, or cloud mode) |
| **Detections** | View history — list view, full preview with details, map view |
| **Rules** | Manage whitelist/blacklist with smart grouping |
| **Stats** | Charts and statistics by day, week, month, year, or custom range |
| **Settings** | Account, sync, webhook, detection thresholds, region |
