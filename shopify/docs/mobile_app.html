📱 App Overview

MareArts ANPR is a professional license plate recognition app for parking management, security checkpoints, and vehicle tracking. All AI processing happens on-device for privacy and speed.

🧭 Navigation

The app has 5 main tabs at the bottom:

Tab Purpose
📷 Scan Fast on-device plate scanning
🕐 Detections View history and saved plates
✅ Rules Manage whitelists/blacklists
📊 Stats View statistics and charts
⚙️ Settings Configure app and sync

📷 1. Scan Page

Purpose: Fast and accurate on-device license plate detection and recognition

Camera Controls:

  • Flash 🔦 - Toggle camera flash
  • Zoom - Pinch or tap to zoom (1x-5x)
  • Camera Switch - Front/back camera toggle
  • Tap to Focus - Tap screen to focus on plate

Scan Modes:

  • Single Capture (⭕) - Capture one plate at a time
  • Continuous Mode (🔄) - Auto-scan continuously
  • Cloud Mode (☁️) - Server-side processing
  • Swipe left/right - Quick mode switching

Camera Settings NEW

Tap the ⚙️ icon next to mode buttons:

Resolution:

  • 1080p (1920×1080) - Highest quality (~140-160ms)
  • 720p (1280×720) - Balanced ⭐ Recommended (~100-130ms)
  • 480p (640×480) - Fastest (~80-110ms)

Frame Rate:

  • 60 FPS - Smooth preview (default)
  • 30 FPS - Battery saver mode

💡 Tip: Lower resolution = faster processing and less storage!

Trial Mode:

  • 100 scans per day (free)
  • Counter shows: "Today: X/100"
  • Resets daily at midnight
  • Login for unlimited scans

🕐 2. Detections Page

Purpose: View all captured license plates

List View

  • All plates grouped by date (Today, Yesterday, etc.)
  • Shows plate number, time, location, thumbnail
  • Swipe to change status (whitelist/blacklist)
  • Tap to view full details
  • Status badges: Green/Red/Orange

Full Preview (tap any detection)

  • Large plate image
  • Edit plate number (✏️ icon)
  • Detection & OCR confidence
  • Date, time, GPS location
  • Copy, Delete, Add to lists buttons

Map View

  • All plates shown on interactive map
  • Clusters nearby detections
  • Tap to zoom in or view details
  • Satellite/Road toggle
  • Search and filter by plate

Export Data NEW

Tap ⋮ menu in top-right:

  • Export All Data (CSV) - Download all detections
  • Includes plate numbers, timestamps, GPS, confidence scores
  • Compatible with Excel, Google Sheets, marearts.com
  • Perfect for backup, analysis, or reporting

💡 Tip: Export regularly to keep backups!

✅ 3. Rules Page

Purpose: Manage plate whitelists and blacklists

List Types:

  • Whitelist (Green) - Allowed/approved vehicles
  • Blacklist (Red) - Blocked/unwanted vehicles

Smart Grouping NEW

  • Organized by letter - Rules grouped in A-Z sections
  • Tap to expand/collapse - Show or hide each group
  • Easy navigation - Quick scroll to find any plate
  • Perfect for large lists - Works great with thousands of rules

Management:

  • + Button - Add new plate
  • Swipe left to delete
  • Search bar - Filter plates in real-time
  • Auto-uppercase, partial matching

Download Rules from Web NEW

Upload rules on marearts.com, download on your phone:

  1. Go to marearts.com/my-account
  2. Upload your CSV file with rules
  3. Open app → See blue banner "Rules package ready"
  4. Tap to download → Choose replace or add
  5. Done! Rules imported automatically

💡 Benefits: Upload large CSV files from computer, share between team, bulk import thousands of rules!

CSV Import/Export

Tap ⋮ menu for more options:

  • Export All Rules - Download as CSV for backup
  • Import Rules - Upload CSV directly from phone
  • Download Sample CSV - Get template with instructions

📊 4. Stats Page

Purpose: View scanning statistics and trends

Features:

  • Overview - Total scans, today, this week, month, year
  • Top 10 Vehicles - Most frequent plates (tap to view all scans)
  • Time Period Charts - Week/Month/Year bar charts
  • Status Filter - View by whitelist/blacklist/unknown
  • Pull to refresh - Update data

⚙️ 5. Settings Page

Purpose: Configure app and cloud sync

Account:

  • Login - Enter email + signature for unlimited scans
  • Trial Mode - 100 scans/day (resets at midnight)

Cloud Sync - Background Mode NEW

Tap "Sync Now" for two-way synchronization:

  • Runs in background - Keep using the app while syncing
  • Progress shown - See sync status in real-time
  • Step 1: Upload your changes to cloud
  • Step 2: Download data from other devices/web
  • Step 3: Merge automatically

What Gets Synced:

  • ✅ All detections (plate numbers, timestamps, GPS)
  • ✅ All rules (whitelist & blacklist)
  • ✅ Detection images from all devices
  • ✅ Changes from marearts.com web

See what's synced and when

Benefits:

  • 📱 Use multiple devices with same account
  • 🌐 View all data on marearts.com
  • 🔄 Rules added on web appear on all devices
  • 💾 Cloud backup of all your data

💡 Tip: Sync runs in background - no need to wait!

Detection Settings:

  • Detection Threshold (60-95%) - Minimum confidence to detect plate (Recommended: 90%)
  • OCR Threshold (60-95%) - Minimum confidence for text recognition (Recommended: 90%)
  • Max Detections (1-10) - Plates per scan (Recommended: 1)
  • Ignore Duplicate (0-60s) - Prevent re-saving same plate (Default: 5s)
  • Plate Region 🌍 - Universal, Europe+, Korea, North America, China

Storage:

  • Save Images 📷 - Store full-resolution images
  • Clear All Data 🗑️ - Delete detections and rules
  • Factory Reset 🔄 NEW - Fresh start, cloud backup stays safe
  • Data Retention - Auto-delete old records (Default: 30 days)

Location:

  • Enable GPS 📍 - Save location with detections (required for Map view)

🎯 Common Workflows

Parking Management:

  1. Add residents to Whitelist (Rules page)
  2. Scan vehicles entering (Scan page)
  3. Check status - Green = Allowed, Red = Blocked
  4. Review violations (Detections page)

Security Checkpoint:

  1. Add approved vehicles to Whitelist
  2. Add banned vehicles to Blacklist
  3. Use Continuous scan at entrance
  4. Audio/vibration alerts for blacklist

Vehicle Tracking:

  1. Scan vehicles continuously
  2. View history in Detections
  3. Use Map to see locations
  4. Export data as CSV for analysis

💡 Tips & Best Practices

For Best Detection:

  • Distance: 2-3 meters from vehicle
  • Angle: Perpendicular to plate
  • Lighting: Good outdoor light (daytime)
  • Focus: Tap plate area to focus
  • Stability: Hold steady while capturing

Settings Recommendations:

High Accuracy (parking/security):

  • Detection: 90% | OCR: 90%
  • Max: 1 plate | Region: Specific
  • Resolution: 1080p or 720p

High Speed (traffic monitoring):

  • Detection: 70% | OCR: 70%
  • Max: 5 plates | Region: Universal
  • Resolution: 480p

🔒 Privacy & Security

  • 100% On-Device: All AI processing on your device
  • No Auto-Upload: Images never uploaded unless you sync
  • Local Database: All data stored locally
  • GPS Optional: Can disable location tracking
  • Your Control: Cloud sync only when you choose
