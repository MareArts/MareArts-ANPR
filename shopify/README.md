# Shopify Assets

HTML files for marearts.com Shopify store.

## Structure

```
shopify/
├── liquid_templates/     # (gitignored) Shopify Liquid templates
│
└── docs/                 # Product page documentation HTML
    ├── app_guide.html    # Mobile app user guide
    ├── mobile_app.html   # Mobile app product description
    ├── anpr_sdk.html     # ANPR SDK documentation
    └── anpr_sdk_v2.html  # ANPR SDK documentation (updated)
```

## Usage

### Documentation HTML
Paste into Shopify product page description (HTML mode)

## Note

`liquid_templates/` is excluded from git (contains payment integration code).
